# Male-Female-Classifier
In this project, I will explore the Male and Female Dataset scrapped from multiple e-commerce websites and build a model that can predict their corresponding gender.
This project requires the following libraries :
•	Torch(Pytorch)
•	Numpy
•	Cv2(OpenCV)
•	Selenium
•	Matplotlib
Please ensure you have installed the following libraries mentioned above before continuing.

## INSTRUCTIONS TO CHECK PREDICTION

To check predictions Extract all files in one folder
Run CMD in that directory
and type
python prediction.py --img_root "C:/user/data/img.jpg"
